## Installation
To run the project locally, follow these steps:

## Clone the repository:

```bash
git clone https://github.com/Riser17/NikeCloneApp.git.git
```

Navigate to the project directory:

```bash
cd NikeApp
```

## Install dependencies:

```bash
npm install
npm start
```

## Dependencies
- React Native
- Redux
- @expo/vector-icons
